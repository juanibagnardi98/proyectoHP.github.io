# Proyecto Hogwarts - Web Estática

Sitio web institucional sobre el mundo mágico de Harry Potter, con 5 páginas: inicio (quiz), personajes, hechizos, casas e historia.

## Tecnologías
- HTML5
- SASS
- Bootstrap 5
- Animate.css
- JavaScript

## Estructura
- `index.html`: Página del quiz interactivo
- `personajes.html`: Información sobre personajes
- `hechizos.html`: Lista de hechizos
- `casas.html`: Descripción de las casas de Hogwarts
- `historia.html`: Historia del mundo mágico
- `sass/`: Archivos SASS
- `css/styles.css`: CSS compilado
- `jib/MAIN.JS`: Script del quiz

## Instalación
1. Clona el repositorio
2. Instala dependencias: `npm install`
3. Compila SASS: `npm run build-css`
4. Abre `index.html` en el navegador

## Despliegue
- Sube a GitHub
- Despliega en Vercel o Netlify conectando el repo
